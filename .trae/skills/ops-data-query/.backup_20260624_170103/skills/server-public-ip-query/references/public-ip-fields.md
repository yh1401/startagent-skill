# 服务器公网IP查询字段映射表

## 请求参数字段

| Key | 中文名称 | 类型 | 是否必须 | 说明 |
|-----|---------|------|----------|------|
| id | 记录ID | string | 非必须 | 公网IP记录ID |
| pageNo | 页码 | int | 非必须 | 默认1 |
| pageSize | 每页大小 | int | 非必须 | 默认40 |
| belong | 归属类型 | int | 非必须 | 如2 |
| ip | 内网IP | int | 非必须 | IP筛选条件 |
| node | 节点名称 | string | 非必须 | 如"省公司->新疆乌鲁木齐" |
| publicip | 公网IP | int | 非必须 | 公网IP筛选 |
| netType | 网络类型 | int | 非必须 | 如1 |
| type | 类型 | int | 非必须 | 如4 |
| isCheckPass | 是否审核通过 | int | 非必须 | 如1 |
| delFlag | 删除标志 | int | 非必须 | 如0 |
| shareBandwidthId | 共享带宽ID | string | 非必须 | 共享带宽标识 |
| elbId | ELB ID | string | 非必须 | 负载均衡ID |
| elbPrivateIp | ELB内网IP | string | 非必须 | 负载均衡内网IP |

## 返回参数字段

| Key | 中文名称 | 类型 | 说明 |
|-----|---------|------|------|
| 公网ip | 公网IP地址 | string | 公网IPv4地址 |
| 公网ipv6 | 公网IPv6地址 | string | 公网IPv6地址 |
| ip | 内网IP | string | 服务器内网IP |
| vip | 虚IP | string | 虚拟IP地址 |
| 机房 | 所在机房 | string | 机房名称 |
| 内网映射端口 | 内网端口 | string | 内网映射的端口号 |
| 公网映射端口 | 公网端口 | string | 公网映射的端口号 |
| 共享带宽id | 共享带宽ID | string | 共享带宽标识 |
| 带宽 | 带宽大小 | string | 带宽值，如100M |
| 带宽类型 | 带宽类型 | string | 独享/共享 |
| 监控计费类型 | 计费类型 | string | 监控计费方式 |
| 公网ipv6备注 | IPv6备注 | string | IPv6相关备注 |

## 用户输入字段 → API参数映射

| 用户可能的中文表达 | 对应API参数 | 值类型 | 示例 |
|-------------------|-------------|--------|------|
| 机房/节点/地域 | node | string | "云公司->贵州" |
| IP/内网IP | ip | string | "192.168.7.101" |
| 公网IP/外网IP | publicip | string | "113.12.13.14" |
| 网络类型 | netType | integer | 1 |

## 支持后置过滤的字段

| 字段 | 中文名称 | 过滤方式 | 示例值 |
|------|---------|---------|--------|
| 带宽 | 带宽大小 | 数值比较 | > 50M, = 100M |
| 带宽类型 | 带宽类型 | 值匹配 | = '独享', = '共享' |
| 机房 | 机房名称 | 模糊匹配 | ~ '贵州' |
