# CMDB 服务器查询参数转换完整指南

**⚠️ 本文件是 CMDB 服务器查询参数转换规则的唯一权威来源！**

本文档定义了从用户自然语言查询到 CMDB API 参数的完整转换规则。

---

## 一、概览

### 1.1 职责

定义 CMDB 服务器查询的参数转换规则和映射表，将用户自然语言查询转换为 CMDB API 请求参数。

### 1.2 配置文件说明

| 文件 | 用途 | 说明 |
|------|------|------|
| `config/params.yaml` | 代码运行时读取的配置文件 | 根据本文档规则生成，与本文档保持同步 |
| `config/api.yaml` | API 接口配置 | 环境配置、端点地址等 |
| `SKILL.md` | 技能说明文档 | 引用本文档作为参数转换规则来源 |

### 1.3 接口信息

| 属性 | 值 |
|------|------|
| **接口地址** | `POST /api/v2/cmdbServer/getCmdbServerBaseMessageList`（开发环境）或 `POST /api/v2/cmdbServer/getCmdbServerPageList`（生产环境） |
| **Content-Type** | `application/json` |

## 二、字段总览

CMDB API 共支持 **20个查询字段**，按处理方式分为 5 类：

| 分类 | 字段数 | 处理方式 | 说明 |
|------|-------|---------|------|
| **固定可选值映射** | 6 | 技能内直接映射 | 状态、归属、系统类型、服务器类型、XC、国产化替换 |
| **字符串模糊匹配** | 4 | 原值传递 | 主机名、IP、资产编号、序列号 |
| **机房节点映射** | 1 | 关键词映射 | 常见节点直接映射，其他原值传递 |
| **复合查询** | 1 | 组合解析 | nodeIpStr（机房+IP组合） |
| **文档可选值映射** | 7 | 参考文档模糊匹配 | 产品、项目、负责人、机架、合同类型（需维护可选值文档） |

## 五、接口可选值字段处理方案

### 5.1 处理流程

以下 7 个字段的值需要通过外部参考文档来维护可选值列表。处理逻辑如下：

```
用户输入 → 关键词提取 → 匹配字段类型 → 到参考文档模糊匹配 → 按选择模式构造参数
```

**关键规则**：
- **单选字段**：用户提到一个名称，文档中匹配到一个结果 → 直接传该结果对应的 ID/名称
- **多选字段**：用户提到多个名称，文档中匹配到多个结果 → 传数组格式
- **模糊匹配规则**：用户输入的关键词与文档中的"显示名称"做包含匹配（like），不要求完全一致
- **匹配失败**：如果用户输入无法匹配到文档中的任何值，则**跳过该字段**，不输出到参数中

### 5.2 文档可选值映射表

| API参数 | 中文 | 参考文档来源 | 传值方式 | 选择模式 | 匹配逻辑 |
|---------|------|-------------|---------|---------|---------|
| productId | 产品 | product-options.md | 传id | 单选 | 用户输入产品名称 → 文档匹配 → 取对应id |
| projectIdList | 项目 | project-options.md | 传id数组 | 多选 | 用户输入项目名称列表 → 逐个匹配 → 取id数组 |
| rack | 机架号 | rack-options.md | 传名称 | 单选 | 用户输入机架号/位置 → 文档匹配 → 取对应名称 |
| operABList | 负责人A或B | person-options.md | 传名称数组 | 多选 | 用户输入负责人姓名 → 匹配到任一字段即加入数组 |
| operAList | 负责人A | person-options.md | 传名称数组 | 多选 | 用户输入负责人姓名 → 匹配到operA即加入数组 |
| operBList | 负责人B | person-options.md | 传名称数组 | 多选 | 用户输入负责人姓名 → 匹配到operB即加入数组 |
| contractType | 合同类型 | contract-options.md | 传id | 单选 | 用户输入合同类型名称 → 文档匹配 → 取对应id |

### 5.3 文档格式建议

每份参考文档使用统一的 Markdown 表格格式，便于技能读取和匹配：

**示例（product-options.md）**：
```markdown
# 产品可选值列表

| 产品名称 | 产品ID |
|---------|--------|
| 规则引擎平台 | 02b970e590224317b7231766f22f6190 |
| 数据中台 | 1a2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p |
| AI平台 | 2b3c4d5e6f7g8h9i0j1k2l3m4n5o6p7q |
```

**示例（person-options.md）**：
```markdown
# 负责人可选值列表

| 姓名 | 所属组 |
|------|--------|
| 刘金鑫 | A组 |
| 姚烨 | B组 |
| 张三 | A组 |
```

### 5.4 匹配示例

| 用户输入 | 匹配字段 | 文档匹配结果 | 生成的参数 |
|---------|---------|------------|-----------|
| "查找规则引擎平台的服务器" | productId | 规则引擎平台 → 02b970e590224317b7231766f22f6190 | `{"productId": "02b970e590224317b7231766f22f6190", "pageSize": 15}` |
| "查找刘金鑫负责的服务器" | operABList | 刘金鑫 → A组 | `{"operABList": ["刘金鑫"], "pageSize": 15}` |
| "查找规则引擎平台和数据中台的服务器" | projectIdList | 规则引擎平台→id1，数据中台→id2 | `{"projectIdList": ["id1", "id2"], "pageSize": 15}` |
| "查找龙晖九楼B01机架的服务器" | rack | 匹配到龙晖九楼B01机架 | `{"rack": "龙晖九楼B01机架", "pageSize": 15}` |

### 5.5 文档维护说明

1. **存放位置**：在技能目录下创建 `references/` 文件夹存放所有可选值文档
2. **更新时机**：当 CMDB 系统中产品、项目、人员等基础数据发生变化时，同步更新对应文档
3. **未匹配处理**：用户输入的关键词如果在文档中匹配不到，跳过该字段，不报错

## 六、完整参数转换示例

### 6.1 仅固定映射字段
```
输入：查找贵州机房的云网在线Linux服务器
输出：{"node": "云公司->贵州", "belong": 2, "state": "0", "osType": "linux", "pageSize": 15}
```

### 6.2 固定映射 + 模糊匹配
```
输入：查找主机名为prod的服务器
输出：{"hostName": "prod", "pageSize": 15}
```

### 6.3 固定映射 + IP查询
```
输入：查找IP为192.168.1.1的在线服务器
输出：{"ip": "192.168.1.1", "state": "0", "pageSize": 15}
```

### 6.4 固定映射 + 文档可选值
```
输入：查找规则引擎平台的在用服务器
输出：{"productId": "02b970e590224317b7231766f22f6190", "state": "0", "pageSize": 15}
```

### 6.5 全字段综合示例
```
输入：查找贵州机房视联的在线Linux信创服务器，主机名为prod
输出：{"node": "云公司->贵州", "belong": 2, "state": "0", "osType": "linux", "isFromOutside": "1", "hostName": "prod", "pageSize": 15}
```

### 6.6 全部查询
```
输入：查找所有服务器
输出：{"pageSize": 100}
```

## 七、参数约束

| 参数 | 类型 | 约束条件 | 选择模式 |

| node | string | 常用值直接映射，其他原值传递 | 单选 |

| ip | string | 原值传递 | - |
| hostName | string | 原值传递 | - |
| assetNo | string | 原值传递 | - |
| serial | string | 原值传递 | - |
| nodeIpStr | string | 格式："机房名称 IP地址" | - |
| productId | string | 文档匹配，传id | 单选 |

| rack | string | 文档匹配，传名称 | 单选 |
| operABList | array | 文档匹配，传名称数组 | 多选 |
| operAList | array | 文档匹配，传名称数组 | 多选 |
| operBList | array | 文档匹配，传名称数组 | 多选 |
