# 产品查询字段映射表

## 请求参数字段

| Key | 中文名称 | 类型 | 是否必须 | 说明 |
|-----|---------|------|----------|------|
| id | 产品ID | string | 非必须 | 产品唯一标识 |
| pageNo | 页码 | int | 非必须 | 默认1 |
| pageSize | 每页大小 | int | 非必须 | 默认40 |
| orderBy | 排序字段 | string | 非必须 | 排序字段名 |
| name | 产品名称 | string | 非必须 | 产品名称 |
| flag | 标志 | int | 非必须 | 1启用，0禁用 |

## 返回参数字段

| Key | 中文名称 | 类型 | 说明 |
|-----|---------|------|------|
| 产品名 | 产品名称 | string | 产品名称 |
| 产品主贡 | 产品主贡 | string | 产品主要贡献 |
| 上级产品 | 上级产品 | string | 上级产品名称 |
| 产品级别 | 产品级别 | string | L1/L2/L3等 |
| 启用 | 启用状态 | string | 启用/禁用 |
| 所属单位/部门 | 所属部门 | string | 产品所属部门 |
| 产品经理 | 产品经理 | string | 产品经理姓名 |
| 运维负责人 | 运维负责人 | string | 运维负责人姓名 |

## 用户输入字段 → API参数映射

| 用户可能的中文表达 | 对应API参数 | 值类型 | 示例 |
|-------------------|-------------|--------|------|
| 产品ID | id | string | "prod-001" |
| 产品名称/产品名 | name | string | "规则引擎平台" |
| 启用/启用状态 | flag | integer | 1 |

## 支持后置过滤的字段

| 字段 | 中文名称 | 过滤方式 | 示例值 |
|------|---------|---------|--------|
| 产品级别 | 级别 | 值匹配 | = 'L1', = 'L2' |
| 所属部门 | 部门 | 模糊匹配 | ~ '基础平台' |
| 产品经理 | 经理 | 模糊匹配 | ~ '张' |
