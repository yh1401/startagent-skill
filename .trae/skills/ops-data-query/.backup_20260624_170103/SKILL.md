# 智能运维数据查询助手

## 概述

本技能提供基于自然语言的运维数据查询能力，通过意图路由将用户查询分发到对应子技能。支持 5 类查询：

| 子技能 | 功能 | 触发关键词 |
|--------|------|------------|
| cmdb-server-query | 查询服务器配置信息 | 服务器、主机、机房、IP、配置 |
| server-public-ip-query | 查询服务器公网IP和带宽 | 公网IP、外网IP、带宽、出口IP |
| project-deployment-query | 查询项目部署记录 | 部署、上线、发布、版本 |
| product-query | 查询产品信息 | 产品、产品线、产品名称 |
| project-basis-query | 查询项目基础信息 | 项目、代码仓库、SVN、GIT |

## 路由规则

### 优先级顺序

| 优先级 | 技能 | 触发条件 |
|--------|------|----------|
| 1 | server-public-ip-query | 明确提到"公网IP"、"外网"、"带宽" |
| 2 | project-deployment-query | 明确提到"部署"、"发布"、"上线"、"部署记录" |
| 3 | product-query | 明确提到"产品"、"产品线"、"产品ID" |
| 4 | project-basis-query | 明确提到"项目"、"代码"、"仓库"、"SVN"、"GIT" |
| 5 | cmdb-server-query | 通用服务器查询（兜底） |

### 意图冲突解决

| 用户输入 | 匹配技能 | 原因 |
|---------|---------|------|
| "XX的服务器" | cmdb-server-query | 核心意图是查服务器 |
| "XX的公网IP" | server-public-ip-query | 核心意图是查公网IP |
| "XX的部署" | project-deployment-query | 核心意图是查部署 |
| "XX的项目" | project-basis-query | 核心意图是查项目信息 |
| "XX的产品" | product-query | 核心意图是查产品 |

### 子技能文件结构

每个子技能 `SKILL.md` 包含：
- **工具调用 Schema**：JSON Schema 格式的参数定义
- **参数映射规则**：自然语言到 API 参数的转换
- **API 接口**：请求/响应格式
- **输出格式**：标准/详细/降级三种模式
- **Mock 数据**：API 不可用时的降级数据

## 通用参数映射

### 机房节点

| 用户输入 | 映射值 |
|----------|--------|
| 贵州、贵阳、贵州机房 | 云公司->贵州 |
| 北京、北京机房 | 云公司->北京 |
| 上海、上海机房 | 省公司->上海 |
| 广州、广州机房 | 云公司->广州 |
| 朝阳、朝阳机房 | 云公司->朝阳1 |

### 服务器状态

| 用户输入 | 映射值 |
|----------|--------|
| 在线、在用、使用中 | "0" |
| 库存、空闲、未用 | "1" |
| 计划上线 | "2" |
| 维修中、维修 | "3" |
| 已报废 | "4" |
| 待分配 | "5" |
| 待清退 | "6" |

### 服务器类型

| 用户输入 | 映射值 |
|----------|--------|
| 物理机、实体机 | "0" |
| 虚拟机、自有虚拟机 | "1" |
| 第三方云机、云机、云服务器 | "2" |

### 部署环境

| 用户输入 | 映射值 |
|----------|--------|
| 测试、测试环境 | 1 |
| 灰度、灰度环境 | 2 |
| 生产、生产环境 | 3 |
| 研发、研发环境 | 4 |

### 部署状态

| 用户输入 | 映射值 |
|----------|--------|
| 成功、部署成功 | 0 |
| 失败、部署失败 | 1 |
| 进行中、部署中 | 2 |
| 待部署、待发布 | 3 |

### 数量词 → pageSize

| 用户输入 | pageSize |
|----------|----------|
| 一台、一个 | 1 |
| 几台、一些 | 15 (服务器) / 40 (其他) |
| 所有、全部 | 100 |
| 最新的、最近的 | 10 |

### 分页指令

| 用户输入 | 操作 |
|----------|------|
| 下一页、继续、更多 | currentPage + 1 |
| 上一页、返回 | currentPage - 1 (最小1) |
| 第N页 | currentPage = N |
| 查看全部 | pageSize = 最大值 |

## 上下文感知

多轮对话中保留以下参数（有效期 5 分钟）：
- 机房位置（node）
- 项目名称（projectName）
- 产品名称（productName）
- 当前页码（currentPage）和每页条数（pageSize）
- 查询模式（simple / detail）

**重置条件**："新的"、"重新"、"重置"，或切换到不同类型的查询。

## 输出格式

### 标准输出

```
## {技能名称}查询结果

**查询条件**：{用户原始查询}
**匹配技能**：{技能名称}
**查询范围**：{中文条件描述}

---

**结果摘要**：共查询到 N 条记录，当前显示前 M 条

---

| 字段1 | 字段2 | ... |
|-------|-------|-----|
| 数据1 | 数据2 | ... |

---

💡 您可以说"查看详细信息"获取更多字段，或"下一页"查看更多结果
```

### 详细输出（用户要求"查看详细信息"时）

完整的字段表格，包含更多列。

### 降级输出（API 失败时）

```
## 查询结果

> ⚠️ 提示：API 接口调用失败，已切换为模拟数据

{模拟数据表格}

**说明**：当前显示的是模拟数据，真实数据需要服务可用时才能获取。
```

### 错误输出

```
## 查询失败

**错误类型**：{参数格式错误/网络错误/无结果}
**错误原因**：{详细原因}
**建议**：{解决建议}
```
