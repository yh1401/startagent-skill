### 9. 分页参数映射

| 用户参数 | API 参数 | 说明 |
|---------|---------|------|
| page | currentPage | 页码 |
| pageSize | pageSize | 每页条数 |
| current | currentPage | 部分 API 用 current |
| size | pageSize | 部分 API 用 size |

**注意**：处理响应时需要兼容两种字段名：
- `data.current` 和 `data.size`（server-public-ip-query, product-query, project-basis-query）
- `data.currentPage` 和 `data.pageSize`（cmdb-server-query, project-deployment-query）

## 错误状态码说明

| code | HTTP 含义 | 技能内处理 |
|------|---------|----------|
| 200 | 成功 | 正常返回 |
| 401 | 未授权 | FATAL → Mock 降级 + 提示联系管理员 |
| 403 | 无权限 | FATAL → Mock 降级 + 提示联系管理员 |
| 404 | 接口不存在 | FATAL → Mock 降级 + 提示接口配置错误 |
| 500 | 服务器错误 | FATAL → Mock 降级 + 提示稍后重试 |
| 其他 | 未知 | WARN → 尝试降级 |
